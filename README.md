# Meine Notizen

Willkommen in meinem zentralen Notiz-Repository! Hier speichere ich alle wichtigen Infos strukturiert als Markdown-Dateien. Perfekt für mich als Informatiker und Fotograf aus Luzern.

## 📁 Ordnerstruktur
```
├── camera/     
├── film empfehlungen/        

├── mathematik/            
├── physik/            
└── programmiersprachen/            
```

## 🚀 So nutzen
1. **Lokal klonen**: `git clone https://github.com/DEINNAME/meine-notizen.git`
2. **Obsidian öffnen**: Den Ordner als Vault laden + Obsidian Git Plugin für Auto-Sync
3. **Bearbeiten**: Neue `.md`-Dateien erstellen, z. B. `touch programmierung/java-oop.md`
4. **Hochladen**: `git add . && git commit -m "Neue Notiz" && git push`

## ⚙️ Tipps
- **Markdown**: `# Header`, `- Listen`, `**fett**` für schöne Formatierung
- **Obsidian**: Backlinks, Graph View für Verknüpfungen
- **gitignore**: `.obsidian/workspace.json` ignorieren für Konfliktfreiheit
- **Sync**: Personal Access Token (repo-Scope) für sicheren Zugriff
